# Tankespår Studio

Tankespår Studio är ett digitalt arbetsrum för ett modernt författarskap.

Projektet samlar:
- Författarskap
- Forskning
- Tankespår (YouTube)
- Affärsplan
- Projektledning
- Idébank
- Föreläsningar

Målet är att skapa ett personligt operativsystem för skrivande, folkbildning och kreativ verksamhet.
